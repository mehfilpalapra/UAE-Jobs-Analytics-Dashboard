# UAE Jobs Analytics Dashboard

An end-to-end Power BI project analyzing hiring trends, salary patterns, skills demand, job distribution, and application behavior in the UAE job market.  
This dashboard is designed for recruiters, HR teams, and business leaders to understand real hiring patterns using data-driven insights.

---

## üìå Project Objective
To build a professional multi-page UAE job market intelligence dashboard that identifies:

- Salary benchmarks  
- High-demand skills  
- Hiring activity by department and location  
- Application performance  
- Salary anomalies or outliers  

---

## üìÅ Dashboard Structure (6 Pages)

### 1. Overview
A complete summary of the dataset including:
- Total Jobs  
- Open Positions  
- Closed Positions  
- Average Salary  
- High-Paying Jobs  
- Low Application Jobs  
- Total Applications  
- Salary Anomaly Count  
- Navigation bar linking to all pages  

---

### 2. Salary and Anomalies
Detailed salary intelligence:
- Salary Distribution Histogram  
- Salary vs Applications Scatter Plot  
- Salary Outlier Detection  
- High-Paying Job Identification  
- Salary Variance (Month-on-Month)  
- Salary Anomaly Flag (DAX-driven)  

---

### 3. Application Analysis
Insights into application behavior:
- Total Applications  
- Average Applications per Job  
- Low Application Job Count  
- Applications vs Salary  
- Applications by Department  
- Application Trend Over Time  

---

### 4. Department Analysis
Department-level hiring insights:
- Total Jobs by Department  
- Number of Departments  
- Highest Hiring Department  
- Total Applications  
- Salary by Department  
- Job Count Distribution  
- Department Hiring Performance  

---

### 5. Location Analysis
Geo-based hiring patterns:
- Total Jobs by Location  
- Unique Locations  
- Top Hiring Location  
- Total Applications by Location  
- Salary by Location  
- Job Distribution Map  

---

### 6. Skills Analysis
Skills-based intelligence:
- Total Unique Skills  
- Most Demanded Skills  
- Skills Frequency Chart  
- Skill-to-Job Mapping  
- Demand Intensity Indicators  

---
